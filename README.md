# Build your own...

- Clone this repo'
- => Clone the "Lyt.Framework" repo' side by side

- Open the solution in Visual Studio 2026 and build.

Tested with VS 2026 and Avalonia 11.3.8.
Should likely work with Rider, but not tested.
